July 2018 - September 2018

Subreddit Analysis using Natural Language Processing (NLP)

General Assembly

Using NLP techniques to analyze text data within posts from both the Space X and NASA subreddits I have created a production level model that can review a post from within these subreddits and then predict where it originated from. With the public Reddit API I was able to webscrape both of these subreddiits to obtain the data in order to create this model. Using specfiic NLP techniques (CountVectorizer and TfidfVectorizer) I was able to identify the words in the text that held the most influence on identifying which subreddit a post came from. Armed with this information I was then able to apply additional modeling techniques (GridSearch) to my classification model (Random Forest) to predict if a post came from the Space X or NASA subreddit, which performed with very high accuracy.
